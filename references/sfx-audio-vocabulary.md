# SFX and Audio Vocabulary
## Comprehensive Sound Design Reference for AI Video Prompts

> **RULE: Audio is NOT optional.** Unspecified audio causes models to guess, often producing unwanted results (random laughter, wrong music, silence where ambience should be). Always include explicit audio direction in every video prompt.

## Platform-Specific Audio Rules

### Grok 3 (Aurora Engine)
- **Audio generation:** Automatic SFX based on prompt description
- **Format:** Describe sounds naturally within the motion prompt
- **No separate audio block** -- SFX vocabulary weaved into main prompt text
- **Best practice:** Be specific, not generic. Write `"rapid gunfire bursts, shell casings clinking on stone"` not just `"action sounds"`
- **Limitation:** No dialogue/speech generation. Audio is environmental only
- **Example:**
  ```
  Loud rapid gunfire bursts, bullet ricochets on metal shield,
  shell casings clinking on stone ground, dramatic cinematic bass impact.
  ```

### VEO 3.1
- **Audio generation:** Native audio synthesis, generates audio in single pass with video
- **Dialogue format:** `[Character] says: "[dialogue]"` -- MUST use colon syntax
- **Required exclusions:** Always include `"no subtitles, no audience sounds"` in every prompt
- **Dialogue limit:** 12-15 words per 8-second clip
- **Audio block format:**
  ```
  Audio: Quiet modern office ambiance -- soft HVAC hum, distant keyboard typing.
         [Character] says: "Your dialogue here, twelve to fifteen words maximum."
         No background music, no subtitles, no audience sounds.
  ```
- **WARNING:** Unspecified audio causes model to guess. Random "studio audience" laughter is common when audio is not explicitly directed.

### Sora 2
- **Audio generation:** Supports audio but less reliable than VEO for dialogue
- **Dialogue format:** `"[Character]: [dialogue]"`
- **Dialogue limit:** 15-20 words per 10-second clip
- **Best practice:** Keep dialogue minimal. Plan for ADR (Automated Dialogue Replacement) in post.
- **Audio block format:**
  ```
  Sound:
  - Voice: confident, measured pace, clear articulation
  - Ambient: quiet office atmosphere, soft HVAC hum
  - Effects: subtle ambient only, no music
  ```

### Kling 2.5
- **Audio generation:** Limited native audio. Environmental SFX only
- **No dialogue support** -- plan all speech for post-production
- **Format:** Include audio description as part of overall prompt
- **Best practice:** Focus on environmental and atmospheric sounds only
- **Example:**
  ```
  Audio atmosphere: gentle wind, distant birdsong, leaves rustling.
  ```

### Seedance 2.0
- **Audio generation:** Native audio-video sync
- **Dialogue:** Basic speech support, short phrases only
- **Format:** Include audio naturally in prompt description
- **Best practice:** Keep audio descriptions simple and direct
- **Example:**
  ```
  Audio: upbeat electronic music pulse, footstep impacts on concrete,
  ambient city noise in background.
  ```

## Audio Direction by Mood

### Authority / Professional
```
Audio: quiet confident space, soft HVAC hum, minimal ambient.
       No nervous energy sounds, no music, no audience sounds, no subtitles.
```

### Tension / Suspense
```
Audio: low tension drone, charged silence punctuated by subtle environmental creaks.
       Heartbeat pulse barely audible. No music, no subtitles.
```

### Warmth / Invitation
```
Audio: warm pleasant ambiance, soft room tone, possibly gentle positive music fade.
       Inviting conversational atmosphere. No subtitles.
```

### Excitement / Energy
```
Audio: upbeat energy, positive atmospheric sounds, dynamic environmental activity.
       Bright, forward-moving sonic palette. No subtitles.
```

### Contemplation / Reflection
```
Audio: minimal ambient, contemplative silence, soft distant environmental sounds.
       Space for thought. No music, no subtitles.
```

### Urgency / Action
```
Audio: tension-building sounds, rapid environmental activity, sharp impacts.
       High energy sonic landscape matching visual intensity. No subtitles.
```

### Sadness / Vulnerability
```
Audio: quiet stillness, distant ambient only, possibly single emotional piano note.
       Space and emptiness in the soundscape. No subtitles.
```

### Triumph / Victory
```
Audio: orchestral crescendo building, environmental sounds brighten,
       resolving into warm confident atmosphere. No subtitles.
```
